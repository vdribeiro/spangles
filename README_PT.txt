-----------
  README
-----------

Para iniciar o jogo deve chamar o predicado 'inicio'.
Ter em atenção que qualquer input superior a 1 caracter precisa de um ponto (.)
para ser executado. Isto deve-se ao facto de instruções de 1 caracter serem feitas
com 'get_code' e de vários caracteres com 'read'.

Tudo é validado, mas se por alguma razão os inputs ficarem baralhados por causa dos
pontos o jogo é reiniciado.
